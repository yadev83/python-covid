This project is a CLI application that gives information about the covid-\/pandemic evolution in France through the use of a public API.\hypertarget{index_autotoc_md1}{}\doxysubsection{Pre-\/requisite}\label{index_autotoc_md1}
To work with this project, you\textquotesingle{}ll need... ~\newline



\begin{DoxyItemize}
\item Python 3.\+7 or above installed ~\newline
 If you have Python installed and PiP ready, you can easily run the following command to install all the project dependencies at once \+: ~\newline
 {\ttfamily pip install -\/r requirements.\+txt} ~\newline
 On the other hand, you can install every requirement one by one \+: ~\newline

\item The \char`\"{}requests\char`\"{} module installed for Python \+: {\ttfamily pip install requests} ~\newline

\item The \char`\"{}tabulate\char`\"{} module installed for Python \+: {\ttfamily pip install tabulate} ~\newline

\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{To start the app}\label{index_autotoc_md2}
Just clone this repo and run ~\newline
 {\ttfamily py \mbox{\hyperlink{app_8py}{src/app.\+py}}} ~\newline
 and check that covid data ! ~\newline


If you wish to run this app on a computer using windows without having a python installation, you can download a release package that contains an executable made out of the source code using py2exe. (WIP) ~\newline

\begin{DoxyItemize}
\item A basic help page when typing {\ttfamily covid -\/h} that gives help about all the commands of the app
\item A main function that gives a table full of data about today\textquotesingle{}s status in every dept when typing {\ttfamily covid} ~\newline

\item A table containing data for a specific dept when typing {\ttfamily covid -\/dept $<$dpt\+\_\+nb$>$} ~\newline

\item A table for a specific date when typing {\ttfamily covid -\/date $<$dd-\/mm-\/yyyy$>$} ~\newline
